# AI Engineering for BFSI - Feb 2026

This folder contains training notebooks for the Vidvatta December cohort's AI Engineering for BFSI program. The material is organized week by week and currently covers Python foundations, prompt engineering, LLM basics, and an introduction to retrieval-augmented generation (RAG).

## Structure

### Week 1

- `python_fundamentals.ipynb`: Module 1 covering core Python fundamentals.
- `advanced_python.ipynb`: Module 2 covering advanced data types and Python tooling.

### Week 2

- `introduction_to_llms.ipynb`: Lesson 2.0 introducing how large language models work from a prompt engineering perspective.
- `prompt_engineering_1.ipynb`: Lesson 2.1 covering prompting fundamentals.

### Week 3

- `prompt_engineering_2.ipynb`: Lesson 3.1 covering advanced prompting techniques.
- `introduction_to_rag.ipynb`: Module 3.2 introducing retrieval-augmented generation (RAG).

## Intended Use

Use these notebooks in sequence by week. Earlier notebooks establish Python and prompting foundations that support the later LLM and RAG material.
